# Passive-Orchestral-Summing-Pack
![IMG_3203](https://github.com/user-attachments/assets/094312e2-02a4-4aa2-af5d-be48db6c4ca8)

Video describing system: https://youtu.be/avfMIS6j09w?si=Rws9fPx7UGtwEoFt
<br>
The Summing Pack is a passive device instead to minimize mic inputs for an orchestral recording. Eight mic lines can be summed down to one, improving mixing board input efficiency. Note, this is passive so it requires a mic pre with high SINAD.
